\# Smartphone Reviews Web App (PHP)



A PHP web application that allows users to browse smartphones, view rankings, and leave reviews.

The application uses JSON files for persistence and focuses on core web development concepts.



\## Features

\- User registration and login (passwords securely hashed)

\- Smartphone listing and individual smartphone pages

\- Rankings based on reviews

\- User profile page

\- Reviews stored and retrieved per smartphone



\## Tech Stack

\- PHP

\- Bootstrap

\- JSON file persistence
