# Smartphone Reviews Web App (PHP)

A PHP-based web application that allows users to browse smartphones, view rankings, and submit reviews.
The project focuses on core web development principles such as server-side rendering, authentication, and data persistence.

## Overview
Users can register and log in, browse a list of smartphones, view individual smartphone pages, and leave reviews.
Rankings are generated based on review data.  
For simplicity and transparency, application data is stored in JSON files.

## Key Features
- User registration and login with secure password hashing
- Smartphone listing and individual smartphone detail pages
- Review submission and retrieval per smartphone
- Rankings page based on user reviews
- User profile page
- Clean, responsive UI using Bootstrap

## Tech Stack
- **PHP** – server-side logic
- **Bootstrap** – styling and layout
- **JSON** – lightweight data persistence
